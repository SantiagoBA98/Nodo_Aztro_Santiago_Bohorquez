    Description:

    Generar dentro del nuevo repositorio (Aztro_Nodo_Nombre en Kebab_Case), un fichero llamado introduction.md donde en formato MarkDown se presenten. (Nombre, breve introducción de quien eres y contacto institucional).

  • Reto #2 Ramas y universos paralelos (Git Branches)

    Description:

    Para este reto vamos a investigar el concepto de ramas en GIT (https://git-scm.com/book/en/v2/Git-Branching-Branches-in-a-Nutshell) (https://www.youtube.com/watch?v=gjKKtQVVCZU), en nuestro repositorio vamos a crear un archivo HTML en donde contaremos un cuento (utiliza las etiquetas html aprendidas en los subtemas 1.1 y 1.2 de HTML), y haz un commit en tu branch principal (master), después genera una nueva rama a partir de esta y cambia la historia (cambia nombres de personajes, sucesos, etc...) sube la rama a tu respositorio remoto y haz un merge de estas dos ramas.
